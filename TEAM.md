## 📋 Proceso de Trabajo Semanal

- Asignación de tareas
- Creación de ramas para la semana

### **Martes a Jueves - Desarrollo**
- Trabajo individual en ramas
- Commits diarios
- Comunicación de bloqueos

### **Viernes - Integración**
- Pull Requests
- Code Review
- Merge a develop
